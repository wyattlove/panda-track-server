Summary

This page demonstrates the standard buttons used in the application. The themes are based on the standard application styles derived from resources such as reset.css, theme.css, and button.css.


Rules

  • Buttons must always be floated. That is, you must apply a style of float:left, or float:right. Without this the buttons will not behave correctly.
  • Buttons should always include the class: "button". Additional styles can be applied to indicate positive or negative emphasis, as well as which button is the primary.
  • Buttons should always be wrapped inside a div with the class "button". The primary reason for this is to allow for simple and consistent layout. Because the buttons use the margin attribute to ~wiggle when clicked, any use of margins for the purpose of layout should be applied to the wrapping div.
  • There is currently no way to truly center buttons. However, we can reasonably approximate a centered button by right-aligning a button with a container set to 50% width. An example of this is included below.
