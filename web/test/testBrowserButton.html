This demonstrates how to handle in-browser navigation using the "back" and "forward" buttons. The intent is not to prevent the user from leaving the site; nor are we attempting to address any security issues. The sole purpose is to make the site more user friendly. This test is 100% client side. The browser should not make any server side requests. This has been successfully tested with ~current versions of firefox, ie, and chrome in 2012.
