This page demonstrates how css affects the table element. Unlike many of the other tests, this page does NOT include or reference any external stylesheets or tools.

In each example we use a RGB (red, green, blue) borders for the table, row, and cells respectively. However, in most (all?) cases, one or more of the borders do not actually display due to the rules for html or css.

The html structure is identical for all of the sample, only the css differs.

As of Aug 2012, these tests appears consistent between current versions of IE 9, Firefox 14, and Chrome 21.
