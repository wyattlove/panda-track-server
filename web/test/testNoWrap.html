This demonstrates how to right-align child divs, while also disallowing wrapping, but without disabling overflow. The parent disables wrapping and aligns the children to the right. The children then specify a display style of inline-block so that they respect the nowrap and alignment in the parent. Finally, you need to specify a fixed height for the parent, so that it does not expand if children try to display in the overflow.

 

This is a useful solution for top level headers where you want to right align some things like dropdown menus for the user/logout.
